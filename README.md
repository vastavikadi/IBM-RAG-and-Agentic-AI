## IBM RAG and Agentic AI

This repository is a collection of resources and tools for developers. It includes code samples, documentation, and best practices for building AI applications. The goal is to provide a comprehensive guide for developers looking to leverage AI technologies in their projects.

- Based on Course: "IBM RAG and Agentic AI" available on Coursera.

This whole course consists of 10 different courses, each focusing on different aspects of AI and its applications. The repository is structured to align with the course content, making it easier for learners to follow along and apply what they learn in practical scenarios.

## Courses Included

- Course 1: Develop Generative AI Applications: Get Started - 10 hours

- Course 2: Build RAG Applications: Get Started - 7 hours

- Course 3: Vector Databases for RAG: An Introduction - 9 hours

- Course 4: Advanced RAG with Vector Databases and Retrievers - 8 hours

- Course 5: Build Multimodal Generative AI Applications - 8 hours

- Course 6: Fundamentals of Building AI Agents - 11 hours

- Course 7: Agentic AI with LangChain and LangGraph - 11 hours

- Course 8: Agentic AI with LangGraph, CrewAI, AutoGen and BeeAI - 13 hours

- Course 9: Build AI Agents using MCP - 10 hours

- Course 10: RAG and Agentic AI Capstone Project - 14 hours

Building AI applications requires a solid understanding of the underlying technologies and best practices. This repository serves as a valuable resource for developers looking to enhance their skills in AI development.
